  • 01 Java 并发机制的底层原理：volatile 的实现原理、synchronized 实现原理与应用、原子操作实现原理等。

  • 02 Java 内存模型基础：认识理解 Java 内存模型（JMM）、指令重排序与顺序一致性、happens-before。

  • 03 三个同步原语 volatile、锁、final：volatile 内存语义、锁的内存语义、final 域的内存语义。

  • 04 并发编程理论基础：并发编程优缺点、线程与进程、死锁问题等。

  • 05 Java 多线程基础：创建线程的 4 种方式，线程停止、休眠（sleep）、礼让（yield）、插队（join），线程状态、优先级等。
