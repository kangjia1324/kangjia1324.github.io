我的在线简历弄好了，记录 Github Page 使用中的一次填坑！

搞了在线简历，一个静态的 HTML 项目。托管到 Github Pages。

项目地址：我的个人简历

将项目 push 到 Github 仓库后，点进项目的 Settings -> Pages 下，配置好如下图：

按理说，打开 https://kangjia1324.github.io/resume/ 是可以访问了，可是却失败了，404。再次访问 https://kangjia1324.github.io/resume/index.html，却可以正常显示页面。说明 Github Pages 未能自己定位到 index.html。

折腾了一番，究其原因竟然是：
