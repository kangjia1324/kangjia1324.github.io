00 Redis 安装

Linux 安装 Redis-6.0.6

  1. 下载安装包 redis-6.0.6.tar.gz

    http://www.redis.cn/download.html

  2. 解压 Redis 的安装包

    tar -zxvf redis-6.0.6.tar.gz

  3. 进入解压后的文件，可以看到 Redis 的配置文件 redis.conf

  4. 安装基本的环境

  5. 进入 redis 的默认安装路径 /usr/local/bin

  6. 在该目录下新建一个文件夹，将 redis 配置文件，复制到文件夹下

  7. redis 默认不是后台启动的，需要修改配置文件

  8. 回到启动 /usr/local/bin 目录，启动 redis 服务

  9. 使用 redis-cli -p 6379，启动客户端

  10. 查看 redis 的进程是否开启

    ps -ef |grep redis

  11. 如何关闭 redis 服务

  12. 再次查看进程是否存在，不存在了。
