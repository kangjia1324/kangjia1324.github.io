05 Linux vim 简单使用

|字数总计:485|阅读时长:1分钟|阅读量:

05 vi 的三种模式

一般命令模式

vi 打开一个文件，默认就直接进入一般命令模式，也称一般模式。在这个模式中，可以上下左右来移动光标，可以删除字符、删除整行，也可以进行复制、粘贴操作。在这一模式下，有很多命令，如下：

  • 【↑↓←→】：光标上下左右移动一个字符

  • 【Ctrl + F】：屏幕向下移动一页，相当于 【Pagedown】

  • 【Ctrl + B】：屏幕向下移动一页，相当于 【Pageup】

  • 【0】或【Home】：移动到这一行的首字符处

  • 【$】或【End】：移动到这一行最后字符处

  • 【G】：移动到整个文件最后一行

  • 【gg】：移动到这个文件第一行

/word：向光标之下查找名为 word 的字符串

?word：向光标之上查找名为 word 的字符串

n：配合上面两个查找命令使用，继续寻找下一个查找到的字符串。

N：配合上面两个查找命令使用，继续寻找下一个查找到的字符串。

dd：删除光标所在的一整行

ndd：删除光标所在的向下 n 行

yy：复制光标行

nyy：复制光标所在的向下 n 行

p：将复制的数据在光标下一行粘贴

u：恢复前一个操作

【Ctrl + r】：重做上一个操作

.：小数点，重复前一个操作

编辑模式

进入编辑模式按键

i 与 I

a 与 A

o 与 O

r 与 R

【Esc】：退出编辑模式，回到一般命令模式

命令行模式

一般命令模式切换到命令行模式的按键说明

:w：将编辑的数据保存并退出

:q：推出 vi

:q!：若曾经修改，但是不想保持，可强制退出

【:w filename】：将编辑的数据文件保存成另一个文件，类似于 Windows 下的另存为

【:set nu】：显示行号

【:set nonu】：去掉显示的行号
