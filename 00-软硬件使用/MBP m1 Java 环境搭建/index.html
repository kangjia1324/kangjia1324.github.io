MBP M1 Java 与 Maven 下载安装

这是在 Macbook Pro M1 上的第一篇文章，写作工具 Typora。

JDK 安装

JDK 推荐下载 Azul 的，Java 8、11、16 等都有 mac arm 版本，下载速度还挺快的，下载地址。

Maven 安装

  1. 官方下载 tar.gz 版本的安装包，下载后解压，然后可以复制到自己想要存放的目录。

  2. 打开 iTerm，输入命令 vim ~/.zshrc，在文件中键入以下两行代码

  1. wq 保存退出，输入命令 source ~/.zshrc

  2. 验证安装，输入命令 mvn -v

  3. 配置国内阿里云镜像

    打开MAVEN_HOME 目录下的 conf 目录下的 settings.xml 文件，找到 <mirrors> 节点，往里面添加配置代码

  4. 配置本地仓库

    本地仓库的作用：存放项目中的 jar 包。本地的 maven 项目首先会从本地仓库中获取 jar 包，当无法获取指定的 jar 包时，本地仓库会从远程仓库（或者中央仓库）中下载 jar 包，并缓存到本地仓库中。

    默认的缓存库：${user.home}/.m2/repository，如需修改，可直接修改 settings.xml 文件中的 <localRepository> 节点。

  5. mvn help:system

    最后运行这个命令，没报错说明配置好了。
